# Història de Git

Git va ser creat per Linus Torvalds l'any 2005, el mateix creador del nucli Linux.  
Necessitava una eina ràpida, segura i distribuïda per gestionar el codi de Linux després que el sistema que utilitzaven abans deixés de ser gratuït.

L'objectiu de Git és permetre que moltes persones puguin treballar en el mateix projecte sense perdre canvis i mantenint un historial complet de totes les versions.

Avui en dia, Git és el sistema de control de versions més utilitzat del món i és essencial per al desenvolupament de programari professional.
