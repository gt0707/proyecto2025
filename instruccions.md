# Instruccions de Git

## git init
Inicialitza un repositori buit.

## git status
Mostra l’estat dels fitxers modificats.

## git add
Afegeix fitxers a l’índex.
Exemple: `git add .`

## git commit
Guarda els canvis al repositori.
Exemple: `git commit -m "Primer commit"`

## git push
Envia els canvis a GitHub.
